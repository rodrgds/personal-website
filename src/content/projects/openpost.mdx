**OpenPost** is an open-source alternative to tools like <Logo fixed src="typefully" /> **Typefully** and Buffer. I built it because most self-hosted alternatives like <Logo fixed src="postiz" /> **Postiz** were just too heavy and bloated for what I needed. I wanted something that was completely self-contained, where you could run a single <Logo fixed src="golang" /> **Go** binary on your own server without worrying about data leaving your machine or paying for monthly subscriptions.

The app uses a <Logo fixed src="svelte" /> **SvelteKit** frontend, <Logo fixed src="sqlite" /> **SQLite** for local persistence, and can run with <Logo fixed src="docker" /> **Docker**. It currently supports publishing to <Logo fixed src="x" /> **X**, <Logo fixed src="mastodon" /> **Mastodon**, <Logo fixed src="bluesky" /> **Bluesky**, <Logo fixed src="threads" /> **Threads**, and <Logo fixed src="linkedin" /> **LinkedIn**.

This is still in early development but is already functional for scheduling and publishing posts.
