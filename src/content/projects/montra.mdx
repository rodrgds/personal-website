**Montra** is fashion search across stores available in a shopper's region. It supports normal and AI-assisted queries, pasted product links, and image-based discovery, then returns filterable results with sizes, materials, colors, prices, availability, and retailer links.

The web marketplace uses TanStack Start and <Logo fixed src="react" /> **React**, backed by a Hono API and <Logo fixed src="postgresql" /> **PostgreSQL** catalog. Meilisearch handles fast lexical retrieval, while pgvector and CLIP power visual similarity. The ingestion pipeline normalizes products and images from many retailers without making checkout or exact stock promises Montra cannot support.
