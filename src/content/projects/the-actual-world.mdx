**The Actual World** was my **Professional Aptitude Project** for high school. It consists of a mobile app ([the-actual-world/mobile](https://github.com/the-actual-world/mobile)) and a separate landing/admin site ([the-actual-world/landing](https://github.com/the-actual-world/landing)).

The mobile product was built with <Logo fixed src="expo" /> **Expo** / **Expo Router** + <Logo fixed src="react" /> **React Native**, <Logo fixed src="supabase" /> **Supabase**, <Logo fixed src="postgresql" /> **PostgreSQL**, <Logo fixed src="stripe" /> **Stripe**, maps, QR codes, and push-notification tooling. The landing site and backoffice were built separately in <Logo fixed src="php" /> **PHP** / <Logo fixed src="mysql" /> **MySQL**, with multilingual content, download/waitlist/contact flows, and admin analytics.

The technical setup involved push notifications, QR code generation for secure connections, and a custom backoffice in PHP for analytics.
